# MERN ThinkBoard

A full-stack note-taking application built with the MERN stack. It features a responsive UI, a robust REST API, and real-world concepts like rate limiting using Upstash Redis.

## üöÄ Live Demo

Check out the deployed application here: **[https://mern-note-5bor.onrender.com](https://mern-note-5bor.onrender.com)**

## ‚ú® Features

- **CRUD Operations:** Create, Read, Update, and Delete notes seamlessly.
- **Responsive UI:** Built with React, Vite, Tailwind CSS, and DaisyUI for a modern look on all devices.
- **Rate Limiting:** Implemented using Upstash Redis to prevent API abuse (10 requests per 20 seconds).
- **Real-time Feedback:** Uses `react-hot-toast` for user notifications.
- **RESTful API:** A clean backend structure using Express.js and Mongoose.
